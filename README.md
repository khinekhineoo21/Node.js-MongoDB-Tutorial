# tutorial_prj

## Project setup

create ```mongodb.yml``` in config folder like as ```momgodb.yml.sample```

```
npm install
```

```
# run for server at local
npm run start
```
```
# run for server at test state
npm run start:test
```
```
# run for server at develop state
npm run start:dev
```
```
# run for server at production state
npm run start:prod 
```
